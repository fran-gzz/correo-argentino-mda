# Prompt para Gamma AI — Presentación Portal MDA

## Instrucciones generales

Generá una presentación en español (Argentina) de 18 a 20 diapositivas con tono institucional formal, pensada para capacitar a operadores N1 de nuevo ingreso en el uso del Portal de Mesa de Ayuda (MDA) de Correo Argentino.

## Identidad visual

- **Colores institucionales:**
  - Amarillo primario: `#FFC72C`
  - Azul secundario: `#254888`
  - Blanco/gris muy claro para fondos
- **Tipografía:** Sans-serif moderna, limpia y legible
- **Estilo:** Profesional, institucional, con acentos en amarillo y azul
- **Formato:** 16:9 widescreen

## Indicaciones generales

- Cada diapositiva debe tener un **propósito claro**: explicar PARA QUÉ SIRVE cada herramienta, no solo mostrar botones
- Incluir **placeholders visibles** donde el usuario deba insertar capturas de pantalla reales del sitio, indicando QUÉ captura debe ir allí (ej: "[INSERTAR CAPTURA: Pantalla de login del portal MDA]")
- Usar íconos representativos para cada módulo
- Incluir animaciones sutiles de entrada entre diapositivas
- Texto en español con lenguaje formal pero accesible para un operador nuevo
